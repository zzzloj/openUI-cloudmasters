Form Manager

File Information

Description: Setup multiple forms with the ability to for pm, email or topic alerts when new form submissions are made. Also included is the ability to add custom fields for each form.

Important

Before you ask for support, make sure you have read the full instructions included below and completed any file edits or hook/application installations required.

Support can be found from DevFuse.com or where you downloaded this modification from.

Step 1: Upload Required Files

Included with these instructions is a folder called upload, upload the contents of this folder to your forums corresponding folders, keeping the folder struction intact.

Step 2: Application Installation

Open the Admin CP > System Tab > Applications & Modules > Manage Applications & Modules page. Navigate to the 'Applications Not Installed' section, browse and locate the 'Form Manager' entry, then click the Install link. Follow the prompts until the application is successfully installed. Leave the Duplicate Tables dropdown unchanged.

Step 3: Using Application

Open your Admin CP > Other Apps Tab Dropdown > Form Manager. From here you can setup your forms and custom fields.
Open your Admin CP > Members Tab > Manage Member Groups > select group > Form Manager. From here you can setup your member permissions.
