Board Width Hook - Documentation

1) Installation

Install Hook

Navigate to your Admin Control Panel, in the left sidebar, look for Applications & Modules. Click on 'Manage Hooks'

Click 'Install Hook', browse to the download package and double click tctc91_boardwidth.xml. Simply click 'Install'.

2) Configuration

To configure your board width, navigate to System Settings -> Hooks -> Board Width by Tom Christian


Enter either a fixed (px) or fluid (%) width. EG - 1024px or 90%.
